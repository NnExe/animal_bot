## Введение
Проект animal_bot представляет из себя телеграмм-бот для отправки изображений в ответ на введенную команду

## Функционал проекта:
- Посылает кошечек :) 
- Посылает собачек :)

## Используемые технологии
При создании и разворачивании приложения использовались следующие технологии:
- ```python 3.7```
- ```python-telegram-bot 13.12```
- ```requests 2.28.0```
- ```docker```

## Переменные окружения
В директории infra репозитория находятся файл ```all.env```, содержащий следующие переменные окружения:\
```TOKEN``` секретный токен для телеграмм-канала\
При необходимости измените значения.

## Запуск приложения
Для запуска приложения перейдите в директорию infra и введите 
```
docker-compose up
```
